# Intelligent Email Threat Detection System

## Project Overview

This mini project focuses on detecting potential email threats such as phishing, spam, and malicious content. It also provides user awareness messages to educate users about safe email practices.

## Features

* Email content analysis
* Threat detection (phishing/spam indicators)
* Rule-based or ML-based classification (based on your implementation)
* User awareness messages
* Simple and interactive output

## Technologies Used

* Python
* NLP / Machine Learning (if used)
* Regex / Rule-based logic

## How It Works

1. User inputs email content
2. System analyzes keywords/patterns
3. Detects whether email is safe or suspicious
4. Displays warning + awareness message

## How to Run

```bash
python main.py
```
